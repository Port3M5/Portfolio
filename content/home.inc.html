Name: PrPl
URL: PrPl Homepage
Description:

My final year project in LSBU, PrPl is a project management tool built in Ruby on Rails 3. It allows Users to be assigned to groups and groups can be assigned tasks to complete, these tasks can then be used to generate GANTT charts automatically, so you can see the state of project completion automatically. It is currently in the design stage.
